## ⚙️ Vim 的三种核心模式

Vim 主要在以下三种模式之间切换：

| 模式名称 | 作用 | 切换方式 |
| :--- | :--- | :--- |
| **1. 命令模式 (Normal Mode)** | 浏览文件、执行复杂操作（复制、粘贴、删除、撤销）、进入其他模式。 | 按 `Esc` 键 |
| **2. 插入模式 (Insert Mode)** | 像普通编辑器一样输入和修改文本。 | 从命令模式通过 `i`, `a`, `o` 等键进入 |
| **3. 末行模式 (Last Line / Command-Line Mode)** | 执行保存、退出、查找、替换等命令。 | 在命令模式下按 `:` 键进入 |

-----

## 🚀 基础操作步骤

### 步骤 1: 启动 Vim 并打开文件

在终端中输入 `vim` 加上文件名即可打开或创建文件。

```bash
vim <文件名>
```

打开文件后，您默认处于 **命令模式**。

### 步骤 2: 进入插入模式 (开始编辑)

在 **命令模式** 下，您需要按下以下键之一来进入 **插入模式**，以便开始输入文本：

| 命令 | 作用 |
| :--- | :--- |
| **`i`** (insert) | 在**光标当前位置**前插入文本。 |
| **`a`** (append) | 在**光标当前位置**后追加文本。 |
| **`o`** (open line) | 在光标所在行的**下方**打开一个新行并插入文本。 |
| **`A`** (Append end of line) | 快速跳转到行尾并插入。 |

### 步骤 3: 退出插入模式 (停止编辑)

当您完成文本输入后，需要回到 **命令模式** 才能进行保存、退出或执行其他命令。

| 命令 | 作用 |
| :--- | :--- |
| **`Esc`** | 随时按下 `Esc` 键，即可从插入模式回到 **命令模式**。 |

### 步骤 4: 保存、退出和查找（末行模式）

在 **命令模式** 下，按下 `:` 键，底部会出现一个冒号，表示进入了 **末行模式**。

| 命令 | 作用 |
| :--- | :--- |
| **`:w`** | 保存 (write) 文件，但不退出 Vim。 |
| **`:q`** | 退出 (quit) Vim。如果文件有未保存的更改，会报错。 |
| **`:wq`** | **保存并退出** (最常用)。 |
| **`:q!`** | **强制退出**，**不保存**任何更改（用于放弃修改）。 |
| **`/<text>`** | 查找指定文本。输入文本后按 `Enter`。按 `n` 查找下一个，`N` 查找上一个。 |
| **`:s/old/new/g`** | 替换命令。在**当前行**全局 (`g`) 将 `old` 替换为 `new`。 |
| **`:%s/old/new/g`** | 替换命令。在**整个文件** (`%`) 全局 (`g`) 将 `old` 替换为 `new`。 |

## ⌨️ 常用编辑操作（命令模式）

在 **命令模式** 下，您可以高效地移动光标和修改文本。

| 操作 | 命令 | 说明 |
| :--- | :--- | :--- |
| **光标移动** | `h` (左), `j` (下), `k` (上), `l` (右) | Vim 的经典移动键。 |
| **删除行** | **`dd`** | 删除光标所在行。 |
| **删除字符** | **`x`** | 删除光标所在位置的单个字符。 |
| **删除到词尾** | `dw` | 删除从光标位置到词尾的内容。 |
| **复制行** | **`yy`** | 复制（yank）光标所在行。 |
| **粘贴** | **`p`** | 在光标下方粘贴内容。 |
| **撤销** | **`u`** | 撤销上一个操作 (undo)。 |
| **重做** | **`Ctrl + r`** | 重做 (redo) 被撤销的操作。 |

## 🌟 总结重点

1.  **开始编辑：** `i`（插入模式）
2.  **停止编辑：** `Esc`（回到命令模式）
3.  **保存退出：** `:wq`（末行模式）
4.  **放弃退出：** `:q!`（末行模式）
